# 更新日志

## v1.1.3 - 2026-06-30

### 新增

- 新增 `latest.json`，用于上传到 Gitee 作为国内备用更新通知源。
- 程序在 GitHub 自动更新失败或便携版手动检查更新时，会尝试读取 Gitee 备用更新源。

### 优化

- 备用更新提示中提供 GitHub、百度网盘和夸克云下载入口。
- 更新说明中补充 Gitee `latest.json` 的上传和 Raw 链接说明。

## v1.1.2 - 2026-06-30

### 调整

- 更新提示中新增 GitHub Releases 下载入口。
- 侧边栏链接区移除百度网盘和夸克云，只保留常用项目链接。
- 当 GitHub Release 缺少 `latest.yml` 时，改为显示中文说明和下载入口，不再展示长串技术错误。
- 新增 Gitee `latest.json` 备用更新通知源，改善国内用户无法访问 GitHub 时收不到更新提示的问题。

## v1.1.1 - 2026-06-30

### 调整

- 默认打包命令 `npm run dist` 改为只生成安装版，避免便携版打包时因为文件占用导致失败。
- 便携版禁用应用内更新，并在设置与说明文件中提示需要手动下载新版。

## v1.1.0 - 2026-06-30

### 新增

- 支持在壁纸卡片上勾选多个壁纸。
- 支持一键全选或全不选当前显示的壁纸列表。
- 支持把多个已选壁纸一次添加到同一个收藏夹。
- 支持把多个已选壁纸文件夹移动到用户指定的其他文件夹。
- 支持启动时自动检查新版本，并在应用内选择下载和重启安装更新。

### 优化

- 批量操作按钮和提示文案改为中文。
- 设置中新增“检查更新”入口和更新状态显示。
- 移动壁纸文件夹时会自动避开重名文件夹。
- 移动后会自动清理原目录中的收藏夹和喜欢状态记录。

### 修复

- 防止把壁纸移动到当前库目录本身。
- 防止把壁纸移动到已选壁纸文件夹内部。
- 支持跨磁盘移动壁纸文件夹。

## v1.0.0 - 初始版本

- 支持选择本地 Wallpaper 壁纸目录并扫描壁纸。
- 支持预览、搜索、排序、打开和删除壁纸文件夹。
- 支持修改应用内显示名称。
- 支持喜欢壁纸和创建收藏夹。
- 支持收藏夹模糊预览和基础个性化设置。
