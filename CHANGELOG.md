# 更新日志

## v1.4.5 - 2026-07-03

### 新增

- 软件启动后会自动检查是否有新版本。
- 安装版启动时优先使用 GitHub 自动更新检查；如果检查失败，会自动尝试 Gitee 备用更新源。
- 便携版启动时会检查 Gitee 备用更新源，发现新版本时提示用户手动下载。

### 优化

- 启动检查更新时，如果没有新版本，不会弹窗打扰用户。
- 只有检测到新版本时，才显示更新提示和下载入口。
- 保留设置中的“检查更新”按钮，用户仍然可以随时手动检查。

## v1.4.3 - 2026-07-02

### 新增

- 新增 Steam 创意工坊状态检测，可通过本地文件夹名中的 Workshop ID 访问对应 Steam 页面。
- 检测前可选择范围：当前列表、已选壁纸、全部本地壁纸、指定来源文件夹，以及未检测、已可访问、疑似下架、需要登录、请求过快、检测失败等状态。
- 检测状态会保存到本地设置，下次打开软件或刷新库时无需重复检测即可看到上次结果。
- 检测间隔可在弹窗中设置，最低 10 秒，减少触发 Steam 限流的概率。
- 检测过程中新增进度弹窗和进度条，完成后显示汇总提示。
- 可识别 Steam 页面中的官方标题，并在壁纸可访问时选择将应用内显示名更新为官方标题。

### 优化

- 将“需要登录”和“请求过快”从“疑似下架”中拆分出来，减少误判。
- 检测结果会在卡片上以不同颜色展示，便于快速筛选本地残留壁纸。
- 保留本地文件夹名作为 Workshop ID，不会因为更新显示名而影响后续检测。
- 清理主进程中部分损坏的旧提示文案，避免编码异常导致启动失败。

## v1.2.0 - 2026-07-01

### 新增

- 支持添加多个壁纸来源文件夹，并在列表中合并展示。
- 支持按来源文件夹单独或组合筛选显示壁纸。
- 收藏夹改为全局共享，不同来源文件夹里的壁纸可以加入同一个收藏夹。

### 优化

- “选择文件夹”入口改为“添加文件夹”，更符合追加多个来源的使用方式。
- 批量移动跨来源壁纸时，确认弹窗会按来源文件夹汇总显示移动数量。
- 旧版本按目录保存的收藏夹会自动迁移并按同名合并为共享收藏夹。

## v1.1.3 - 2026-06-30

### 新增

- 新增 `latest.json`，用于上传到 Gitee 作为国内备用更新通知源。
- 程序在 GitHub 自动更新失败或便携版手动检查更新时，会尝试读取 Gitee 备用更新源。

### 优化

- 备用更新提示中提供 GitHub、百度网盘和夸克云下载入口。
- 更新说明中补充 Gitee `latest.json` 的上传和 Raw 链接说明。

## v1.1.0 - 2026-06-30

### 新增

- 支持在壁纸卡片上勾选多个壁纸。
- 支持一键全选或全不选当前显示的壁纸列表。
- 支持把多个已选壁纸一次添加到同一个收藏夹。
- 支持把多个已选壁纸文件夹移动到用户指定的其他文件夹。
- 支持启动时自动检查新版本，并在应用内选择下载和重启安装更新。

### 修复

- 防止把壁纸移动到当前库目录本身。
- 防止把壁纸移动到已选壁纸文件夹内部。
- 支持跨磁盘移动壁纸文件夹。

## v1.0.0 - 初始版本

- 支持选择本地 Wallpaper 壁纸目录并扫描壁纸。
- 支持预览、搜索、排序、打开和删除壁纸文件夹。
- 支持修改应用内显示名称。
- 支持喜欢壁纸和创建收藏夹。
- 支持收藏夹模糊预览和基础个性化设置。
