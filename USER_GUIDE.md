# 壁纸盒使用手册

<img src="assets/logo.png" alt="壁纸盒 Logo" width="180" />

壁纸盒用于浏览和管理仍保留在本地磁盘中的 Wallpaper Engine 壁纸资源。它不会上传壁纸文件，也不需要开启 Wallpaper Engine。

## 开始使用

1. 打开壁纸盒后，点击“选择文件夹”。
2. 选择 Wallpaper Engine 的壁纸保存根目录。常见位置类似：`SteamLibrary\steamapps\workshop\content\431960`。
3. 壁纸盒会读取一级子文件夹内的 `preview.jpg`、`preview.png` 或 `preview.gif`，并以卡片方式显示。GIF 会保持动态预览。

所选路径会被记住，下次打开会自动读取。

## 浏览与管理壁纸

- 使用顶部搜索栏按显示名称筛选。
- 可按修改时间、文件夹大小或名称排序。
- 在“布局”中选择自动或 2–6 列卡片布局。
- 心形按钮用于标记或取消“喜欢”。
- 右上角书签按钮用于加入或移出收藏夹。
- 名称右侧的笔形按钮可修改应用中的显示名称；真实文件夹和文件不会被改名。
- “打开”会在资源管理器中打开实际文件夹。
- “删除”会删除对应的整个本地壁纸文件夹，且不可恢复。

## 收藏夹与模糊预览

在侧边栏点击“设置”上方的收藏夹管理入口，或顶部“新建收藏夹”，可创建收藏夹。

- 创建时可以勾选“在此收藏夹中模糊预览图”。
- 右上角“应用收藏夹模糊”默认开启。
- 当某壁纸属于任一设置为模糊的收藏夹时，它在主页、喜欢列表和收藏夹视图中都会被模糊显示；关闭该开关即可临时查看清晰预览。
- 删除收藏夹只会删除分类关系，不会删除任何壁纸文件。

## 个性化设置

侧边栏的“设置”可调整并保存：

- 背景色和背景图片
- 主题色
- 界面透明度
- 字体大小和导入字体
- 本地背景音乐

所有设置均保存在本机。点击“恢复默认”可清除这些个性化设置。

## 注意事项

- 本应用只识别包含 `preview.jpg`、`preview.png`、`preview.jpeg` 或 `preview.gif` 的一级子文件夹。
- 重命名显示名称、喜欢与收藏夹均为壁纸盒自己的记录，不会写入 Wallpaper Engine。
- 删除前请确认不再需要对应资源；删除操作会移除该壁纸目录内的所有文件。

## 更新说明

- 安装版支持在应用内检查、下载并重启安装更新。
- 便携版不支持应用内更新。如果使用便携版，请在发布新版后手动下载新的便携版程序。
- 更新提示中会提供 GitHub、百度网盘和夸克云下载入口，便携版用户可通过这些入口手动下载新版。
