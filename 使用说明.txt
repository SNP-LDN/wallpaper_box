壁纸盒 exe 使用说明
====================

软件名称：壁纸盒 / wallpaper_box
用途：浏览和管理本地残留的 Wallpaper Engine 壁纸资源。


一、这个软件是做什么的？
----------------------

当 Wallpaper Engine 里的壁纸被删除、下架或屏蔽后，软件内可能已经找不到它们，
但本地磁盘里仍然会残留对应文件。

壁纸盒可以帮助你：

1. 扫描本地 Wallpaper Engine 壁纸文件夹。
2. 可视化查看每个壁纸文件夹里的 preview.jpg、preview.png 或 preview.gif。
3. 不打开 Wallpaper Engine，也能浏览、搜索、收藏、标记喜欢和删除本地壁纸资源。
4. 给壁纸设置自己的显示名称，不会修改真实文件夹名。

注意：壁纸盒只读取你选择的本地文件夹，不会上传壁纸文件。


二、下载后怎么打开？
------------------

如果你下载的是安装版：

1. 双击 wallpaper_box-版本号-setup.exe。
2. 按照安装提示继续。
3. 安装过程中可以选择安装位置。
4. 安装完成后，可以从桌面快捷方式或开始菜单打开“壁纸盒”。

如果你下载的是便携版：

1. 双击 wallpaper_box-版本号-portable.exe。
2. 不需要安装，打开后即可使用。
3. 适合临时使用、测试，或放在 U 盘里使用。


三、第一次使用
--------------

1. 打开壁纸盒。
2. 点击“选择文件夹”。
3. 选择 Wallpaper Engine 的本地壁纸保存目录。

常见路径类似：

D:\SteamLibrary\steamapps\workshop\content\431960

如果你的 Steam 安装在 C 盘，也可能类似：

C:\Program Files (x86)\Steam\steamapps\workshop\content\431960

选择后，壁纸盒会读取这个目录下的一层子文件夹，并寻找：

- preview.jpg
- preview.jpeg
- preview.png
- preview.gif

找到预览图后，会用卡片形式显示出来。GIF 会保持动态显示。


四、主要功能怎么用？
------------------

搜索：
在顶部搜索栏输入名字，可以筛选显示的壁纸。

排序：
可以按修改时间、大小、文件名排序。

布局：
可以选择自动布局，也可以选择固定列数显示。

喜欢：
点击卡片右下角的心形按钮，可以标记或取消喜欢。
被喜欢的壁纸可以在侧边栏“喜欢”里快速查看。

收藏夹：
点击卡片右上角的书签按钮，可以把壁纸加入一个或多个收藏夹。
如果壁纸已经加入收藏夹，书签会显示为橙黄色。

新建收藏夹：
点击“新建收藏夹”，会弹出收藏夹窗口。
你可以查看现有收藏夹，也可以输入新收藏夹名称。
创建时可以选择这个收藏夹内的壁纸是否模糊预览。

删除收藏夹：
删除收藏夹只会删除分类关系，不会删除任何壁纸文件。

修改显示名称：
点击壁纸名称旁边的笔形按钮，可以修改它在壁纸盒里的显示名称。
这只会修改壁纸盒里的显示，不会修改真实文件夹名。

打开：
点击“打开”会在资源管理器中打开对应的本地文件夹。

删除：
点击“删除”会删除对应的本地壁纸文件夹。
这个操作会移除该文件夹内的文件，请确认后再删除。


五、模糊预览
------------

收藏夹可以设置“模糊预览”。

如果一个壁纸属于启用了模糊的收藏夹，并且顶部的“应用收藏夹模糊”已开启，
那么这个壁纸在主页、喜欢页和收藏夹视图里都会被模糊显示。

默认情况下，“应用收藏夹模糊”是开启的。


六、设置
--------

侧边栏里可以进入设置页面。

你可以修改：

- 背景图片
- 背景颜色
- 背景音乐
- 界面透明度
- 字体
- 字体大小
- 软件主题颜色

这些设置会保存下来，下次打开软件时仍然生效。

如果想恢复最初样式，可以点击“恢复默认”。


七、用户数据保存在哪里？
----------------------

壁纸盒会在你电脑的应用数据目录里保存自己的设置，例如：

- 上次选择的壁纸路径
- 修改后的显示名称
- 喜欢列表
- 收藏夹
- 个性化设置

这些数据不会写进 Wallpaper Engine 的文件夹，也不会上传到网络。

常见位置类似：

C:\Users\你的用户名\AppData\Roaming\壁纸盒

开发版运行时也可能保存在：

C:\Users\你的用户名\AppData\Roaming\wallpaper-local-library


八、注意事项
------------

1. 壁纸盒只识别一层子文件夹里的 preview 图片。
2. 修改显示名称不会修改真实文件夹名。
3. 删除壁纸文件夹是本地删除，请谨慎操作。
4. 如果选择文件夹后没有显示内容，请确认选择的是：

   steamapps\workshop\content\431960

   而不是更上层的 SteamLibrary 或 workshop 文件夹。

5. 如果图片没有显示，可能是该壁纸文件夹内没有 preview.jpg、preview.png 或 preview.gif。
