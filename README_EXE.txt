壁纸盒 exe 使用说明
====================

一、安装版和便携版
------------------

安装版文件名通常类似：
wallpaper_box-版本号-setup.exe

安装版需要先安装，然后从桌面快捷方式或开始菜单打开。安装版支持在应用内检查、下载并重启安装更新。

便携版文件名通常类似：
wallpaper_box-版本号-portable.exe

便携版不需要安装，双击即可运行，适合临时使用、测试或放在 U 盘里使用。

重要说明：便携版不支持应用内更新。如果你使用便携版，更新提示会提供 GitHub、百度网盘和夸克云入口，请手动下载新的便携版程序，或者改用安装版。

二、推荐选择
------------

普通用户推荐下载并使用安装版。安装版更适合长期使用，并且可以收到新版本提示。

便携版只推荐临时使用。便携版不会自动更新，也不会在应用内完成更新安装。

四、国内备用更新源
------------------

如果 GitHub 访问不稳定，程序会尝试读取 Gitee 上的 latest.json 来显示更新通知。

请把项目里的 latest.json 上传到 Gitee 仓库，并确认它可以通过 Raw 链接直接访问。
